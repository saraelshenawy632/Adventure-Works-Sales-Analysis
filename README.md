# Adventure-Works-Sales-Analysis


Adventure Works Sales Analysis | Excel & Power Query
📌 Project Overview

The Adventure Works Sales Analysis project focuses on transforming raw sales data into meaningful business insights using Microsoft Excel, Power Query, and Pivot Tables. The project includes data cleaning, transformation, KPI analysis, and the development of an interactive dashboard to monitor sales performance and support data-driven decision-making.

🎯 Objectives
Analyze overall sales performance.
Track key sales KPIs and trends.
Identify top-performing products and categories.
Evaluate customer and regional sales patterns.
Provide actionable insights for business growth.
